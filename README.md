Project 6: NYC Taxi Trip Duration Prediction

Dataset: NYC Taxi Trip Duration Dataset (Kaggle)

Problem Statement:
Predict the duration of taxi trips in New York City based on pickup and drop-off locations, timestamps, passenger count, and trip distance. Analyze traffic patterns and temporal factors affecting trip duration.

Tasks Performed:
•	Data cleaning and removal of outliers
•	Feature engineering using pickup and drop-off timestamps
•	Distance calculation using latitude and longitude
•	Model training using Linear Regression and Random Forest
•	Evaluation using RMSE and R-squared metrics

Skills and Tools Used:
Pandas, NumPy, Feature Engineering, Machine Learning, Scikit-learn

Real-World Application:
Urban transportation analytics and ride-hailing optimization
